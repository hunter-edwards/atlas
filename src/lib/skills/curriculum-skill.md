# Curriculum Design Skill

## Module Design

A module is a thematic unit covering one major topic area. Each module should:

- Have a clear, measurable learning objective (e.g., "Students will be able to explain supply and demand dynamics")
- Follow a logical concept sequence: foundations → intermediate ideas → synthesis
- Estimated duration: 1-3 weeks depending on complexity
- End with a module exam that tests deeper understanding

## Lesson Design

Each lesson covers a single concept or tightly related set of concepts. Follow this arc:

1. **Hook** (1 slide): Why this matters. A question, surprising fact, or real-world scenario.
2. **Concept** (3-5 slides): The core idea, explained clearly with definitions and context.
3. **Example** (2-3 slides): Concrete, real-world examples that illustrate the concept.
4. **Application** (1-2 slides): How to use this knowledge. Thought experiments, practice scenarios.
5. **Summary** (1 slide): Key takeaways, connection to next lesson.

### Slide Count Targets
- Standard lesson: 8-12 slides
- Dense/technical lesson: up to 15 slides
- Review lesson: 5-8 slides
- Session duration target: 45-60 minutes

## Quiz Design

### Cadence
- **Lesson check**: After every lesson. 3-5 questions. Quick comprehension verification.
- **Module exam**: After each module. 8-12 questions. Tests deeper understanding and connections.
- **Final exam**: End of course. 15-20 questions. Comprehensive, draws from all modules.

### Question Type Distribution
- 60% Multiple Choice (4 options each)
- 25% Short Answer (1-3 sentence responses)
- 15% True/False

### Writing Good Distractors (Wrong MC Answers)
Good distractors should:
- Be plausible — they represent common misconceptions or partial understanding
- Be similar in length and style to the correct answer
- Not be obviously absurd or unrelated
- Test a specific misconception, not just be random

### Bloom's Taxonomy Alignment by Difficulty

| Level | Primary Bloom's Levels | Question Style |
|-------|----------------------|----------------|
| 100-level | Remember, Understand | Define, describe, identify, list |
| 200-level | Understand, Apply | Explain, compare, demonstrate, use |
| 300-level | Apply, Analyze | Solve, differentiate, organize, relate |
| 400-level | Analyze, Evaluate | Critique, justify, assess, argue |
| Graduate | Evaluate, Create | Design, synthesize, propose, theorize |
